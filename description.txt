 ---   Git   ---

1. git add . *или файл, который добавляем*
2. git commit -m "..." // Коммит
3. git push origin main // Загрузка в гит
4. git pull origin main // Подтянуть из ветки
5. git rm файл // Удалить файл отовсюду
6. git rm --cached файл // Удалить файл только из гит
